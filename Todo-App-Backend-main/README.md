# Todo App (Basic Full Stack Project)

This is a simple Todo application built using basic web technologies.  
It includes a login system and allows users to manage their daily tasks.

---

## 📌 Features

- User Login (basic authentication)
- Add new tasks
- Update existing tasks
- Delete tasks
- Logout functionality

---

## 🛠 Technologies Used

- HTML
- CSS (basic styling)
- JavaScript (Vanilla JS)
- Node.js
- Express.js

---

## ⚙️ How It Works

1. User logs in using predefined credentials
2. A token is stored in browser localStorage
3. All Todo operations are protected using this token
4. User can perform CRUD operations on tasks
